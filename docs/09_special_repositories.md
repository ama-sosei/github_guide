# 特別なファイル
## README.md
`README.md`はプロジェクトの説明や, 使い方,インストール方法などを理解してもらえるように記述するものです。  
GitHubなどに開発物を公開すると`README.md`はファイルの下に表示されます。  
`README.md`はマークダウン形式のファイルで簡単な構文で記述することができます。
